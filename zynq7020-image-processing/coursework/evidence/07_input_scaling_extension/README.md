# 输入缩放综合扩展证据

本目录是第二周“大拓展”的主线预留位。当前 `main` 暂不合并大拓展实现；实际开发仍保留在 `exp/05-ext-scaling` 分支，等待上机验证和证据补齐。

## 合并前必须补齐

| 材料 | 说明 |
| --- | --- |
| 上位机离线测试 | `stretch`、`letterbox`、`center-crop`、处理尺寸等测试日志 |
| 输入来源或尺寸对比 | 至少两种输入来源或两种原始尺寸 |
| 上板现象 | HDMI 照片或视频截图，能区分不同缩放策略或处理尺寸 |
| 串口/上位机日志 | 发送参数、帧计数、错误信息或成功记录 |
| 修改说明 | PC 端、PS 端、PL 端分别是否修改，以及保持兼容的理由 |

大拓展没有现场材料前，不在 `main` 中标记为完成。
