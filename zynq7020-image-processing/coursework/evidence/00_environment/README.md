# 环境与设备证据

本目录保存课程设计使用环境的快照，便于最终报告说明“在哪个工具链、哪块板卡、哪个串口条件下完成验证”。

## 待补材料

| 材料 | 建议文件名 |
| --- | --- |
| Vivado / Vitis / SDK 版本 | `env_vivado_vitis_version.txt` |
| Python / conda 环境 | `env_python_conda.txt` |
| PC 端依赖导入检查 | `env_host_camera_uart_import.txt` |
| 开发板型号、串口号、连接方式 | `env_board_connection.md` |
| 现场测试日期和操作者 | `env_field_test_record.md` |

环境材料不要求很长，但必须能支撑后续证据的可追溯性。
